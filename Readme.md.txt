# Diabetes Trend Analysis: Pre, During, and Post COVID
## Project Overview
This project analyzes diabetes trends before, during, and after COVID-19 using Power BI.
The goal is to understand how COVID impacted diabetes patterns using real datasets.
## Tools Used
- Python (Pandas, NumPy)
- Jupyter Notebook
- Power BI
- GitHub
## Data Sources
- Pima Diabetes Dataset
- COVID-19 Dataset
## Data Cleaning Steps
- Removed invalid zero values
- Filled missing values with median
- Created age groups
- Generated timeline (2017–2023)
- Aggregated COVID yearly statistics
## Dashboard Features
- Interactive filters (Year & Phase)
- KPI cards
- Trend analysis charts
- Age distribution visualization
## Files Included
- cleaned_data.csv → Clean dataset
- data_cleaning.ipynb → Python cleaning code
- diabetes_covid_dashboard.pbix → Power BI dashboard
